--- title: "Unified Diffusion Model with Multi-Scale Style Infusion" excerpt: "An enhanced diffusion-based font generation framework with unified single-phase training, multi-scale style injection, and structure-aware losses across multilingual scripts.
" collection: portfolio --- This project develops an improved diffusion-based font generation framework that addresses limitations of two-stage pipelines by unifying training into a single phase. The model simultaneously learns style representation and structural generation, enabling more coherent and efficient font synthesis. **Key Highlights:** - Multi-Scale Style Infusion: style vectors injected at encoder, bottleneck, and decoder stages of the U-Net. - Sobel-based structural consistency loss to enforce stroke-level fidelity during denoising. - CLIP-based style loss for perceptual alignment between reference and generated glyphs. - Supports Korean, Chinese, and Latin script generation from minimal reference samples. - Designed for high-fidelity, structure-aware font synthesis across multilingual scripts. 🔗 _Code coming soon — research ongoing._ 📝 _Extended journal manuscript in preparation._
